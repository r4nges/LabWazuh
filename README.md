# 🔐 SOC Lab – Wazuh + Sysmon

Projeto prático de laboratório focado em monitoramento, detecção e análise de eventos utilizando **Wazuh (SIEM/XDR)** integrado ao **Sysmon** em ambiente virtualizado.

---

## 📌 Visão Geral

Este laboratório simula um ambiente real de Security Operations Center (SOC), contemplando:

- Implantação de SIEM
- Integração de endpoint Windows
- Ingestão de logs via EventChannel
- Criação de regras customizadas
- Mapeamento MITRE ATT&CK

---

## 🖥️ Arquitetura

- 1x VM Linux (Wazuh Manager + Dashboard)
- 1x VM Windows (Wazuh Agent + Sysmon)
